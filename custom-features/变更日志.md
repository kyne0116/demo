```
### 日期: YYYY-MM-DD
**类型**: 功能/修复/优化/文档
**简介**: 功能简述

**详细文档**: [对应文档路径]

**变更内容**:
- 详细变更说明

**影响范围**: 
- 影响的模块或功能

**建议操作**: 
- 用户需要执行的操作
```

### 文档链接说明
- 需求文档: `requirements/REQ-YYYYMMDD-功能名称.md`
- 实现文档: `components/组件名称_技术方案.md` 或 `组件名称_配置指南.md`

### 变更类型说明
- **核心规范**: 影响项目基础架构和开发原则的变更
- **功能**: 新增业务功能
- **修复**: Bug修复和问题解决
- **优化**: 性能优化和代码重构
- **文档**: 文档更新和说明完善
